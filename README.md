## 🌟 一、产品概述与定位

**NeoCompanion** 是一款**融合了「桌面虚拟宠物 + 陪伴式监督学习 + 智能连接网关」的长期陪伴型 AI 系统**。

我们希望打破传统 AI 工具“打开-提问-拿结果-关闭”的冰冷单次交互模式，让 AI 以一个可爱、有温度的**桌面悬浮宠物**形态长期存在于用户的学习、工作和创作现场，陪同用户共同生活与成长，成为真正懂你状态的“数字搭子”。

```
+-------------------------------------------------------+
|  Windows / macOS 桌面                                  |
|                                                       |
|   [ 用户正在写代码/写论文 ]                             |
|                                                       |
|         ( ^_^ )  <-- 桌面悬浮宠物 (Live2D / 动态帧动画)  |
|        /  o   o \    语音 TTS 反馈: "加油哦，你已经专注20分钟啦！"|
|                                                       |
+-------------------------------------------------------+
```

---

## 🎯 二、核心产品决策

与传统效率管理工具或重度 Autonomous Agent 不同，NeoCompanion 坚守以下核心设计决策：

1. **桌宠为核心交互形态**：常驻桌面、轻量化、支持拟人化 TTS 语音发声与丰富动作。通过温度感的“仪式感交互”监督学习，而非施加监控压力的效率看板。
2. **温情化的屏幕状态感知**：本地运行极轻量级窗口特征提取，唯一目的是为了感知用户的专注、分心、卡壳或疲劳状态并触发拟人化关怀（如歪头递上虚拟咖啡或揉眼提醒），绝不进行任何商业隐私监控。
3. **“智能网关”型轻量 Agent 架构**：不从零构建繁琐笨重的自主 Agent。复杂长流程跨应用任务代行分发给开源个人 GUI 助手 [OpenClaw](https://github.com/openclaw/openclaw)；日常提问与编程答疑则直连大模型 API 并融入桌宠极简对话流，保持应用轻量。
4. **极客 Hook 与状态提醒中心**：暴露本地安全端口，支持开发人员使用自定义脚本推送编译、服务器监控或智能家居状态。桌宠会对此进行活泼的拟人化语音播报与动作映射。

## 🛠️ 三、核心能力模型

NeoCompanion 的产品能力由浅入深分为以下四个层次：

```
+-------------------------------------------------------+
|  1. Companion Layer (桌宠悬浮交互 / TTS 语音 / 情绪反馈)  |
|-------------------------------------------------------|
|  2. Routine Layer   (番茄钟 / Todo账本 / 天气 / 宠物小日记) |
|-------------------------------------------------------|
|  3. Gateway Layer   (内置原子Agent / OpenClaw / LLM 极简对话) |
|-------------------------------------------------------|
|  4. Hook & System   (本地安全Hook API / 本地状态感知与记忆) |
+-------------------------------------------------------+
```

### 1. 陪伴交互层 (Companion Layer)
* **悬浮桌宠形态**：支持 Live2D 或动态帧动画，可拖拽、置顶、缩放、贴边隐藏。
* **拟人化语音 TTS**：拥有个性声线，用自然语气与你闲聊、吐槽、鼓励与安慰。
* **状态情绪反馈**：配合用户心流，展示戴眼镜读书、敲迷你键盘工作、伸懒腰、气鼓鼓敲屏幕、揉眼打哈欠等细节动画。

### 2. 日常琐碎陪伴层 (Routine Layer)
* **宠物专属番茄钟**：桌宠与用户同步进入专注，专注期间协助屏蔽非紧急打扰并轻声守护：“嘘，我们在闭关修炼哦。”
* **拟人化待办助手**：将 Todo 列表包装为“宠物的任务小账本”，用可爱的看板与声音温柔催促，划掉任务时宠物会拍手欢呼。
* **天气与时间碎碎念**：根据本地气象与时间段主动发声，提供雨天带伞、深夜关怀与午后轻音乐推送。
* **宠物视角小日记**：日终生成一篇用宠物温柔口吻撰写的成长轨迹日记，记录一天的专注与努力。

### 3. 智能网关与大模型对话层 (Gateway Layer)
* **内置原子能力**：直接提供极速本地文件读写、网页搜索总结与剪贴板报错联动。
* **OpenClaw 任务网关**：将复杂跨软件 GUI 操作委托给 [OpenClaw](https://github.com/openclaw/openclaw)，桌宠前端以“宠物正在后台施法”等精美看板优雅展示进度，让复杂 Agent 过程完全可视化、可控化。
* **开发辅助与极简对话**：在桌宠聊天框直连大模型（如 DeepSeek/Claude），支持一键拾取报错文本、极速提供 Markdown 代码块建议，一键复制代码，保持心流不断。

### 4. Hook 与系统安全层 (Hook & System Layer)
* **安全本地多通道 Hook**：本地监听安全端口，接收用户自定义脚本推送的状态（例如：编译成功，桌宠跳舞欢呼；部署失败，桌宠抱头大哭）。
* **自动配置扫描注册**：启动时自动扫描并注入 OpenClaw 等 CLI 配置文件路径，免去繁琐的手动插件配置。
* **半透明审批气泡 (Permission Bubble)**：外部 Agent 执行高危敏感动作（如覆盖文件或危险 bash 命令）时，挂起 HTTP 连接并在桌面上弹出气泡卡片，展示敏感度评分，支持鼠标点击或全局热键（`Ctrl+Shift+Y` 同意 / `Ctrl+Shift+N` 拒绝）进行免死等一键流式审批。
* **本地隐私感知与记忆**：全部屏幕特征提取基于本地计算，不上传云端；内置轻量 SQLite 与 LanceDB 向量库，构建随着相处越发默契的本地专属记忆。

## 📅 四、阶段演进路线 (Roadmap)

### 🚀 Phase 1: 温暖的陪伴桌宠 (Warm Desk-Pet Base)
* **目标**：实现桌面常驻悬浮宠物的 Live2D 动态展示与 TTS 语音播报，打通日常守护闭环。
* **核心功能**：宠物陪伴番茄钟、拟人化 Todo 小账本、日常天气碎碎念、活跃窗口检测（感知专注与严重分心并触发桌宠语音/动作反馈）。

### 🔌 Phase 2: 自定义 Hook 与网关扩展 (Hooks & Gateway Integration)
* **目标**：实现强大的极客扩展生态与轻量开发辅助闭环。
* **核心功能**：
  * 发布安全本地 Hook API，允许通过 Git push 或编译脚本向桌宠推送状态。
  * **自动配置文件扫描注入**（优先支持 OpenClaw 配置自动注册打桩）。
  * **落地“浮动权限审批气泡 (Permission Bubble)”机制**与全局系统热键操作（`Ctrl+Shift+Y` / `Ctrl+Shift+N`）。
  * 完善内置 AI 对话面板，支持一键抓取剪贴板报错与大模型调优代码块。

### 🧠 Phase 3: 智能 Agent 完整联接 (Full Agent Gateway Ecosystem)
* **目标**：全面成为个人工作流的统一网关伙伴系统。
* **核心功能**：
  * 深度打通 OpenClaw 的 GUI 执行网关，在桌宠面板无缝呈现跨软件执行的微观状态。
  * 引入 ONNX 本地嵌入模型与 LanceDB 向量长效记忆，使宠物的性格、声线、建议策略随长期陪伴产生自适应进化。
